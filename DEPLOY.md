# 部署说明

## 一、本地部署
```bash
npm install
npm run dev
```
默认会启动 Vite 开发服务。

## 二、构建测试
```bash
npm run build
```
构建成功后会输出 `dist` 目录。

## 三、测试环境部署思路
1. 本地确认 `npm run build` 成功
2. 将 `dist` 目录上传到测试服务器
3. 使用 Nginx 托管静态文件
4. 后续再接入真实接口和权限体系

## 四、当前阶段建议
先做：
- 本地部署验证
- 页面比例确认
- 任务中心流程确认
- 数据字段和权限口径确认

暂不建议直接把当前版本作为正式生产版对外开放。
