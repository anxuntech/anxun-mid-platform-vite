# 安巡中台原型（Vite + React + TypeScript）

这是面向本地验证和演示的工程版原型，定位是：
- 草料作为前端采集层
- 自有系统作为查看与分析中台
- 面向企业、安全服务商、保险平台、应急局四类角色演示

## 当前版本重点
- 修复任务中心布局挤压问题
- 恢复首页与驾驶舱内容
- 企业样本扩充到 12 家
- 保留角色切换、企业详情、风险评级、隐患闭环、设备展示、任务中心、人员管理

## 本地启动
```bash
npm install
npm run dev
```

## 打包验证
```bash
npm run build
```

## 演示建议
1. 先看总览首页
2. 再切到驾驶舱大屏
3. 切换到保险平台或安全服务商，查看任务中心
4. 查看企业详情、风险评级、隐患闭环

## 当前说明
这个版本适合：
- 本地部署验证
- 方案演示
- 继续拆分组件和接接口
